## gemini.sh

*Easy tool for work with Gemini API in CLI.*

### SET UP
Need to set the env:
```shell
export GEMINI_API_KEY="YOUR_API_KEY"
```
For further use you have to add it to your `~/.bashrc` or `~/.zshrc`.

### USE
- Start:
```shell
./gemini.sh "Question about anything"
```
- Alias (e.g. in `~/.zshrc`):
```shell
alias gemini='~/myscripts/gemini.sh'
```
